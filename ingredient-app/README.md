# Ingredient App

This is a simple web application that allows users to enter an ingredient and fetch related information using an API.

## Project Structure

```
ingredient-app
├── src
│   ├── index.html        # Main HTML file for the application
│   ├── index.js          # Main JavaScript file handling user interactions
│   ├── styles.css        # Styles for the application
│   └── api
│       └── client.js     # API client for making requests
├── package.json          # npm configuration file
└── README.md             # Project documentation
```

## Getting Started

To set up the project locally, follow these steps:

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd ingredient-app
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   npm start
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to view the application.

## Usage

- Enter an ingredient in the input field and submit the form.
- The application will fetch and display relevant information based on the ingredient provided.
